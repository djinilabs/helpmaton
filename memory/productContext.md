# Product Context

## Overview

Helpmaton is a workspace-based AI agent management platform (SaaS) that enables teams and developers to create, configure, and deploy AI agents at scale. The platform provides infrastructure and tools for deploying intelligent assistants through webhooks and APIs.

## Core Value Proposition

- **Workspace Organization**: Organize AI agents and knowledge bases into isolated workspaces
- **AI Agent Management**: Create and configure agents with custom system prompts
- **Knowledge Base**: Upload and manage documents (markdown/text) that inform agent responses
- **Webhook API**: Each agent exposes a webhook endpoint for easy integration
- **Team Collaboration**: Workspace permissions for granular access control
- **Subscription Plans**: Free, Starter, and Pro tiers with different limits

## Use Cases

- Customer support bots with product documentation
- Internal knowledge assistants for teams
- Specialized workflows (technical docs, legal research, etc.)
- API integration for embedding AI capabilities

## Target Users

- Developers building AI-powered applications
- Teams needing intelligent assistants
- Organizations requiring scalable AI agent infrastructure

## Key Features

- REST API for programmatic access
- Webhook endpoints per agent
- Document management with folder organization
- Workspace-based multi-tenancy
- Credit system for usage tracking
- API throttling based on subscription plans
- Streaming support (SSE, Lambda URLs)
- Integrations: Discord, Email OAuth, MCP servers
